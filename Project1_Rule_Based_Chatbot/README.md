# Rule-Based AI Chatbot

## Description
This project is a simple Rule-Based AI Chatbot developed using Python. The chatbot interacts with users by responding to predefined inputs using conditional statements. It demonstrates the fundamentals of conversational AI without using machine learning.

## Features
- Greets the user
- Responds to predefined questions
- Uses if-elif-else conditions
- Interactive command-line interface
- Handles unknown inputs gracefully
- Exits when the user types "bye"

## Technologies Used
- Python

## Concepts Covered
- Variables
- Input/Output
- Conditional Statements
- Loops
- Functions
