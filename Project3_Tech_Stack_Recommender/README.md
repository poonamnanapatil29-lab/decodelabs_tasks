# Tech Stack Recommender System

## Description
This project recommends an appropriate technology stack based on the user's interests and project requirements. It helps beginners understand which programming languages, frameworks, and tools are suitable for different development domains.

## Features
- Accepts user preferences
- Recommends suitable technologies
- Provides technology stack suggestions
- Simple and interactive interface
- Beginner-friendly implementation

## Technologies Used
- Python
- Pandas

## Concepts Covered
- Data Processing
- Recommendation Logic
- User Input Handling
- CSV File Handling
- Decision Making
